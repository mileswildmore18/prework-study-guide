HTML

  • The head element contains information about the webpage
  • The body element represents the visible content show to the user.
  • The section element defines a section in a document that contains thematic content

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • A border assigns a border with a width and colour of your choice
  • A box shadow adds shadow effects around an element's frame with size and colour of your choice

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git clone [url]: retrieves an entire repository from a hosted location via URL
  • git branch: lists the branches your have now
  • git add -A: commits all files in the working branch
  • git commit -m "[message]": Adds a message that we committed to changes
  • git pull origin [branch]:fetches and merges any commits from the tracking remote branch
  • git push origin[branch]:Transmits local branch commits to the remote repository branch

  • For: Performs a task in a loop until told to end
